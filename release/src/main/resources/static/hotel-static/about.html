关于我们

    沈阳贵德科技开发有限公司——成立于一九九八年，位于沈阳市高新技术开发区，是沈阳市科技局种子基金支持单位，公司一直专注于星级宾馆行业计算机软件和智能产品的研究、开发、服务的专业化公司。公司以技术为推动力，以市场为先导，提供以酒店之星管理系统、餐饮管理系统、宾馆智能客控系统为主导的自主产品。向国内市场提供优质的产品及服务。

    公司设有市场营销部、技术培训部、产品开发部、售后服务部、财务部等部门，这些部门构成公司完整的开发、销售、维护平台，为全公司业务稳定发展提供支持。

    经过十几年的不懈努力，我们的用户已经覆盖全国28个省、市、自治州、直辖市。用户总数达到上千家。
