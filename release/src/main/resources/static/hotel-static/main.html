智能客控

  • 迎宾模式：当客人进入房间插卡取电时，廊灯自动开启，贴心的服务让客人倍感亲切
  • 一键总关：一键总关所有灯光，每路灯光可单独控制，解决客人睡觉前逐一关灯的烦恼
  • 智能夜灯：当客人一键总关所有灯以后，房间内所有灯光关闭的同时自动夜灯亮起，给客人贴心的服务
  • 人体感应：当人进入卫生间时，卫生间灯、换气扇开启；客人离开后延时10分钟后关闭，人性化设计，安全节能
  • 电器控制：可对电视、窗帘等进行智能控制
  • 夜光识别：LED暗背光显示、有利于夜间识别，采用平行光原理，不影响客人睡觉
  • 安全稳定：人接触到的为5V安全电压，安全系数高；内置过载保护，可防浪涌电流和防雷击
  • 安装方便：标准的86式墙壁开关外形设计，不需对灯具做任何改动接配件
  • 维修方便：相比客控主机系统，维修更方便，可与普通开关互控，电工即能解决
